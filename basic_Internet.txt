<인터넷의 동작 개념>

클라이언트 <---인터넷---> 서버

도메인 이름 : 사람이 기억하기 쉽도록 이름을 붙임 (ex) http://a.com
IP 주소 : 실질적으로 도메인에 접속했을 때 연결되는 IP
포트(Port) : 0~65535
    -> (ex) http://a.com:80이라고 치면 80번 포트에 연결하는 것
            :80은 생략 가능(코드 내에서 도메인을 80번 포트에 연결하면 됨)

[서버 종류]
    1. 데이터베이스 서버
    2. 채팅 서버
    3. 게임 서버
    4. 웹 서버
